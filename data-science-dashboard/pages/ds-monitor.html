GETTING STARTED:

This is a simple JavaScript application which means you can run this on Chrome without a server. Due to cross-origin policies, you will need to install the 'Allow-Control-Allow-Origin:*' chrome extension before using this dashboard.

NOTE: This is a temporary dashboard for the Data Science team to check the health of our applications and APIs. This will be deprecated as soon as we get a more robust solution in place. Email us at DataScienceSS@coyote.com for more info.
